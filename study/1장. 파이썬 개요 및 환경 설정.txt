# 2. 주피터 노트북 실행법

(1) 셀 실행에 대하여
- shift + enter : 포커스된 셀이 실행, 커서가 다음 셀로 이동
- alt + enter : 포커스된 셀이 실행, 다음 셀 생성 후 다음 셀로 이동
- ctrl + enter : 포커스된 셀이 실행, 포커스 이동 x
- enter : 재편집

(2) 주석 추가
CODE 상태에서 #[내용] 기입

(3) 라인넘버 설정
view > toggle line number (shift + L ; on/off 가능)

# 3. esc + (hot key setting)
- dd : 셀 삭제
- b : 포커스된 셀 아래에 셀 추가
- a : 포커스된 셀 위에 셀 추가
- m : 마크다운 셀(문서 셀)로 변경
- y : 코드 셀(소스 셀)로 변경
- o : 실행 결과 접기/펴기(토글)

# 주피터 노트북에서 cmd 명령어 : !
ex) !dir

# 매직 명령어 (주피터 노트북 자체 명령어 ; 파이썬 x) : %
ex) %ls
