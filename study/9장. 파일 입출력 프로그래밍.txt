9장. 파일 입출력 프로그래밍

- 파일 : txt, pickle(머신러닝 모델 저장, 넘파이 배열 저장), csv, json, h5(딥러닝 모델 저장)

1. txt 파일 입출력

- 작업 종류
	- w : 기존의 파일이 있는 경우 덮어쓰기. 없을 경우 파일 생성.
	- a : 기존 내용을 보존, 파일의 맨 끝에 내용을 이어 붙임. 없을 경우 파일 생성.
	- r : 기본값, 파일을 읽기만 할 때 사용. 없을 경우 FileNotFoundError 에러 발생.
	- x : 파일을 새로 만들어 쓰기 모드로 생성. 기존의 파일이 있는 경우 FileExistsError 에러 발생.

- 파일 형식 지정 - t : 텍스트 모드, 기본값 (.txt, .csv)

- 상대경로 : 코드파일 기준으로 작성
- 절대경로 : 드라이브 위치부터 작성

 ex) 상대경로
	f = open('data/ch09.txt', 'w', encoding='utf-8')

 ex) 파일을 'a' 모드로 열기
	with open('data/ch09.txt', 'a') as f:
    		print('프린트 함수를 사용한 파일 쓰기\n', file = f)

	class Member:
    		def __init__(self, name, age, email, address):
        		self.name = name
        		self.age = age
        		self.email = email
        		self. address = address
        
    		def __str__(self):

3. csv 파일 입출력

- list 또는 dict 데이터로 읽고 쓰기
-데이터 프레임으로 읽고 쓰기 (11장)

3.1 reader

- 데이터 양옆의 쌍따옴표("")와 행 끝에 있는 줄바꿈 문자(\n)를 걸러냄.
- 쉼표(,)를 기준으로 글자를 분리하여 파이썬의 list로 변환.

 ex)
	import csv
	with open('data/ch09_member1.csv', 'r', encoding='utf-8') as f:
    		reader = csv.reader(f) # f를 통해 파일과 연결 후, reader를 통해 CSV 형식을 list로 변환
    		for row in reader: # 주의 : with 구문을 벗어나면 파일이 닫힘.
        		print(row)

 ex)
	with open('data/ch09_member1-cp.csv') as f:
    		reader = csv.reader(f)
    		result = list(reader) # 파일이 닫히기 전, result에 list 작성
	for row in result: # list를 가져왔기 때문에 파일 상태와 관계 없이 for문 구동 가능
    		print(row)

- quoting : CSV 파일의 따옴표 처리 정책을 정하는 역할. 기본값 - QUOTE_MINIMAL

- QUOTE_NONNUMERIC : 파일에서 따옴표가 없는 데이터를 발견하면 실수형 숫자로 변환

 ex)
	import csv
	with open('data/ch09_member1.csv', 'r', encoding='utf-8') as f:
    		reader = csv.reader(f, quoting=csv.QUOTE_NONNUMERIC)
    		for row in reader:
        	print(row)

4. json 파일 입출력

4.1 json dump

- 딕셔너리 리스트 데이터를 json 파일로 저장
- 객체 리스트 데이터를 json 파일로 저장
- 성질: 파이썬 객체를 변환함과 동시에 오픈된 파일 객체(f)에 즉시 기록
- 매개변수: dump(데이터, 파일객체)

 ex) 딕셔너리 리스트를 json 파일로 저장 시
	import json
	with open('data/ch09_member1.json', 'w', encoding='utf-8') as jsonf:
	    	json.dump(data, jsonf, ensure_ascii=False, indent=' ')
                  	                      # indent : json 파일 저장 시 들여쓰기 사용

 ex) 객체 리스트를 json 파일로 저장 시
	class Member:
    		def __init__(self, name, age, email, address):
        		self.name = name
        		self.age = age
        		self.email = email
        		self.address = address
    		def as_dict(self):
        		'객체를 딕셔너리로 바꾸어 반환, json.dump 사용 시 필요'
        		return self.__dict__
    		def __str__(self):
        		return f'이름 :{self.name}, 나이 :{self.age}, 이메일 :{self.email}, 주소 :{self.address}'

	member_list = [Member('홍길동',20,'h1@h.com','신대방'),
              			Member('김길동',30,'h2@h.com','봉천'),
              			Member('신길숙',25,'h3@h.com','신림')]

	# default : JSON으로 변환할 수 없는 객체를 만났을 때, 이를 어떻게 처리할지 알려줌. 함수 지정 가능.
	with open('data/ch09_member2.json', 'w', encoding='utf-8') as jsonf:
    		json.dump(member_list, jsonf, ensure_ascii=False, indent=' ', default=Member.as_dict) # 객체를 딕셔너리 형태로 반환하는 인스턴스 함수 지정
						# ensure_ascii=False : 영어권 및 일부 특수문자 외 문자를 유니코드 변환 없이 그대로 표현

4.2 json load

- 딕셔너리 리스트 변수로 읽기
- 객체 리스트 변수로 읽기

 ex) json 파일을 딕셔너리 리스트로 읽기
	with open('data/ch09_member1.json', 'r', encoding='utf-8') as jsonf:
    		load_data = json.load(jsonf)

	result = set() # 두 딕셔너리의 item이 같은지 item마다 T/F (백업 체크 ; 원본과 read file 비교)
	for d, load_d in zip(data, load_data):
    		result.add(d==load_d)
	result

 ex) json 파일을 객체 리스트로 읽기

	def as_member(dic):
   		return Member(dic.get('name'), dic.get('age'), dic.get('email'), dic.get('address'))

	with open('data/ch09_member2.json', 'r', encoding='utf-8') as jsonf:
    		load_member_list = json.load(jsonf, object_hook=as_member)
