데이터구조 : 여러 데이터를 한 변수에 담는 것이 목적

1. 리스트 : 데이터 타입에 상관없이 0개 이상의 데이터를 순서대로 담는 구조
	1.1 리스트 생성 : [], list(range(from, to, step))
	1.2 기본 정보 조회
		- min(lst), max(lst), sum(lst), len(lst)
		- sorted(lst, reverse=True) : 정렬된 새로운 리스트를 반환 (원본 유지)
		- lst.sort(reverse=True) : 원본 리스트 자체를 정렬된 값으로 변경 (반환값 없음)
	1.3 item 추가
		- append(value) : 리스트 끝에 하나의 요소를 그대로 추가
		- extend([values]) : 리스트 끝에 다른 리스트 등 다수의 요소를 이어 붙임
		- insert(idx, value) : 지정한 인덱스(idx) 위치에 요소를 삽입
	1.4 인덱싱과 데이터 찾기
		- lst[idx] : 인덱스로 요소 참조
		- lst.index(데이터) : 특정 데이터의 위치(인덱스) 반환 (없는 데이터 조회 시 에러)
		- 데이터 in lst : 데이터가 리스트에 존재하는지 여부 판별 (True/False)
		- lst.count(데이터) : 지정한 데이터가 리스트에 몇 개 있는지 개수 반환
	1.5 원하는 item 추출 및 수정
		- 슬라이싱 : lst[start:stop:step]으로 부분 추출 및 슬라이싱 대입으로 수정 가능
		- 리스트 컴프리헨션 : [표현식 for 변수 in 반복가능객체 if 조건] 형식으로 조건별 데이터 추출
	1.6 item 삭제
		- pop() : 가장 마지막 요소를 꺼내고 리스트에서 삭제 (삭제된 데이터 반환)
		- pop(idx) : 지정한 인덱스 위치의 요소를 꺼내고 삭제 (삭제된 데이터 반환)
		- del lst[idx] : 지정한 인덱스 위치의 요소를 즉시 강제 삭제 (반환값 없음)
		- remove(데이터) : 리스트에서 동일한 첫 번째 데이터를 찾아 삭제 (데이터가 없으면 에러)
	1.7 복제 (깊은 복사)
		- 복제할변수 = 원본데이터.copy() (원본과 독립된 새로운 객체 생성)

2. 튜플 : 리스트와 유사하나 생성 후 수정이 불가능한 구조 (읽기 전용)
	- 생성 방법 : () 소괄호 이용 또는 tuple() 함수 사용
	- 특징 : 수정, 추가, 삭제 관련 메서드 및 문법 사용 불가 (데이터 보호 목적)
	- 주의 : 요소가 1개일 때 뒤에 콤마(,)를 붙이지 않으면 일반 스칼라 데이터로 인식함 (예: (1,) 구조 필수)

3. 딕셔너리 : 키(key)와 값(value)의 쌍으로 구성된 자료 구조
	- 특징 : 순서가 없으므로 인덱스를 이용한 번호 참조는 지원하지 않음
	1. 딕셔너리 생성 및 조회 : {} 중괄호 이용
		- dic['key'] : 키로 값을 조회 (존재하지 않는 키 조회 시 에러 발생)
		- dic.get('key') : 키로 값을 안전하게 조회 (존재하지 않는 키 조회 시 에러 없이 None 반환)
	2. 수정, 추가, 제거
		- 추가 및 수정 : dic['key'] = 값 대입 또는 dic.update(딕셔너리) 이용
		- 제거 : del dic[key] 또는 dic.pop(key) 사용 (존재하지 않는 키 제거 시 에러 발생)
	3. 출력 및 반복문 활용
		- print(dic.keys()) : 딕셔너리의 키 목록 반환
		- print(dic.values()) : 딕셔너리의 값 목록 반환
		- print(dic.items()) : 딕셔너리의 (키, 값) 쌍 목록 반환
		- for k, v in dic.items(): 구조로 반복문에서 키와 값을 동시에 활용 가능
	4. 복제 : 복제할변수 = 원본데이터.copy()

4. 셋 : 중복을 허용하지 않고 순서가 없는 집합 자료 구조
	4.1 셋 생성 : set() 함수 사용 (※ 빈 중괄호 {}는 딕셔너리로 인식하므로 세트 생성 시 주의)
	4.2 데이터 추가
		- add(value) : 기본 자료형 데이터 1개를 추가
		- update(자료형) : 리스트, 딕셔너리, 세트 등의 요소를 하나씩 쪼개어 다수 추가 (딕셔너리는 키만 추가됨)
	4.3 데이터 삭제
		- remove(데이터) : 특정 데이터를 찾아 삭제 (데이터가 없으면 에러 발생)
	4.4 집합 연산자
		- & : 교집합
		- | : 합집합
		- - : 차집합

5. 반복문 보조 함수 (enumerate, zip)
	- enumerate(반복가능객체) : 반복문 실행 시 요소뿐만 아니라 현재 순서인 인덱스(idx) 번호를 동시 반환
	- zip(객체1, 객체2, ...) : 여러 개의 자료구조(리스트, 튜플 등)를 동일한 인덱스 요소끼리 짝을 지어 묶어줌
