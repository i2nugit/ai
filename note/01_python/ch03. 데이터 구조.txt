데이터구조 : 여러 데이터를 한 변수에 담는게 목적

1. 리스트 : 타입에 상관없이 0개 이상의 데이터들
	1.1 리스트 생성 : [], list(range(from, to, step))
	1.2 기본 정보 조회
		- min(lst), max(lst), sum(lst), len(lst), 
		- sorted(lst, reverse=True) : 정렬된 lst를 반환
		- lst.sort(reverse=True) : lst가 정렬된 값으로 변경
	1.3 item 추가
		- append(value), extend([values]), insert(idx, value)
	1.4 인덱싱과 인덱싱 찾기
		- lst[idx]
		- lst.index(찾을 데이터) : 없는 데이터를 찾을 경우 error
		- 찾을데이터 in lst  : 찾을데이터가 있는지 여부 반환
		- lst.count(찾을데이터) : 찾을데이터가 몇개인지 숫자 반환
	1.5 원하는 item 추출
		- 슬라이싱
		- 리스트컴프리헨션(원하는 데이터 추출)
	1.6 item 수정 : 인덱싱이나 슬라이싱을 이용
	1.7 item 삭제 : pop(), pop(idx), del lst[idx], remove(데이터)
	1.8 복제(깊은복사) : 복제할변수 = 원본데이터.copy()

2. 튜플 리스트와 유사하나 수정불가
	- () 이용하여 생성, tuple() 함수 사용 가능, 읽기 전용 데이터 구조
	- 수정이 불가하여, 수정, 추가, 삭제 함수 및 문법 사용 불가

3. 딕셔너리 : 키(key)-값(value)으로 구성된 자료 구조
	- 인덱스를 이용한 참조는 지원하지 않음
	1. 딕셔너리 생성 : {}
		- dic.get('key1')
	2. 수정, 추가 : update(), 제거 : del 변수/pop()
	3. 출력 : for문 (key만, 값만, 키-값 출력 가능)
		print(my_dic.keys()) - 키 목록
		print(my_dic.values()) - 값 목록
		print(my_dic.items()) - 키-값 목록
	- list, tuple, set에서의 in연산자 : 요소가 있는지 여부 반환	
	4. 복제 : 복제할변수 = 원본데이터.copy()

4. 셋 : 중복을 허용하지 않는 집함 (순서 없음)
	4.1 셋 생성 : {}, set() ({} (빈 중괄호)는 딕셔너리로 인식)
	4.2 data 추가 : add() - 기본자료형, update() - list, dic, set의 item이 하나씩 추가; dic은 key만 추가
	4.3 data 삭제 : remove()
	4.4 연산자 : & (교집합), | (합집합), - (차집합)

5. enumerate : 반복자 또는 순서 객체로 반복문을 처리할 때 사용하는 함수 (idx 색인)
	- list, tuple, set을 for문에 적용 시 사용
	- zip 함수
