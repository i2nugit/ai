ch07. 객체지향 프로그래밍

1. 객체와 클래스
- 클래스 : 객체를 만들기 위한 틀 (추상적인 공간)
	- 모든 클래스는 object 클래스를 상속받음
	- 정의 문법: class 클래스명(상속받을클래스명): (첫 글자는 대문자)
- 객체 : 클래스의 인스턴스 (실제하는 데이터)

2. 생성자와 소멸자
- 생성자 `__init__(self, 매개변수, ...)`
	- 객체가 생성될 때 자동 실행
	- 인스턴스 변수 초기화 등 생성 시 필요한 코드 포함
	- 생성자가 없을 경우 기본 생성자가 자동 생성
- 소멸자 `__del__(self)`
	- 객체가 소멸될 때 자동 실행 (인스턴스 객체의 레퍼런스 카운트가 0이 될 때)
	- 소멸 시 필요한 코드 포함
- self
	- 함수를 호출한 객체(인스턴스) 자기 자신을 가리킴
	- PEP 8 규칙에 따른 전 세계적인 약속

3. 변수와 메소드(함수)
- 변수의 종류
	- 인스턴스 변수(멤버 변수) : 각 인스턴스마다 독립적으로 가지는 변수
	- 클래스 변수 : 클래스 내부에 선언되며 생성된 모든 인스턴스가 공유하는 변수
		- `[클래스].[클래스변수]` 수정 시 모든 인스턴스에 반영됨
		- `[객체변수].[클래스변수]` 수정 시 해당 객체의 인스턴스 변수로 새로 생성되어 해당 객체에만 반영됨
	- 스태틱 변수(정적 변수) : 클래스 내부에 선언되어 모든 인스턴스가 공유하는 하나의 변수 
		- 파이썬에서는 변수 앞에 언더바 2개`__`를 붙여 비공개 클래스 변수 형태로 활용
- 메소드의 종류
	- 인스턴스 메소드 : self를 첫 번째 매개변수로 받으며, 인스턴스 속성에 접근 가능
	- 클래스 메소드 : cls를 첫 번째 매개변수로 받으며, 클래스 속성에 접근 가능
	- 스태틱 메소드 : 매개변수에 self나 cls를 받지 않으며, 클래스나 인스턴스 속성에 직접 접근하지 않는 독립적인 함수

정의 문법:
	class Class_name(상속 받을 클래스명): (첫글자는 대문자 ex - Person, Game ...)
		'독스트링'
		클래스 변수 = 값
		def __init__(self, [멤버 변수]=[기본값], ...):
			멤버 변수 초기화
		def __del__(self):
			객체 소멸 시 수행할 명령어
		def method_name(self, [매개변수], ...):
			인스턴스 메소드에서 수행할 명령어, 첫번째 매개변수는 self
		@classmethod
		def class_method_name(clz, [매개변수], ...):
			클래스 메소드에서 수행할 명령어
		@staticmethod
		def static_method_name([매개변수):
        		스태틱 메소드에서 수행할 명령어

	var_name = Class_name()
	var_name.method_name()
	Class_name.class_method_name()
	Class_name.static_method_name()
	var_name.__dict__ : 객체의 멤버 변수 내용을 딕셔너리 형태로 반환

4. 상속과 재정의
- 상속 : 클래스를 재사용하는 방법
	- 부모 클래스의 모든 속성과 메소드가 자식 클래스에게 상속됨
	- 정의 문법: class 자식클래스명(부모클래스명):
	- super() : '부모 클래스의'를 의미하며, 부모 클래스의 메소드나 생성자를 호출할 때 사용
