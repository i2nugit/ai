9장 파일 입출력 프로그래밍
- 파일 : txt, pickle(머신러닝 모델 저장, 넘파이 배열 저장), csv, json, h5(딥러닝 모델 저장)

1. txt 파일 입출력 ; 파일 형식 지정 - t : 텍스트 모드, 기본값 (.txt, .csv)
- 작성 문법
	with open('파일 경로','r/w/a','encoding) as f:
	* 쓰기 - 문자변수; f.write()/리스트변수; f.writelines()
	* 읽기 - 문자변수; f.read()/리스트변수; f.readline(), f.readlines()

2. pickle 객체 입출력
- 형식이 있는 텍스트 데이터 불러오기 (.txt → 객체 리스트 변환)
- 피클링 ; 파일 형식 지정 - b : 바이너리 모드 (.pkl)
- 작성 문법
	with open('파일 경로','rb/wb','encoding) as f:
	* 피클링 	; pickle.dump(저장하고 싶은 객체, f)
	* 언피클링 	; pickle.load(f)

3. csv 파일 입출력 : list 또는 dict 데이터로 읽고 쓰기
- 작성 문법
	with open('파일 경로','r/w/a','encoding) as f:
	* reader = csv.reader(f)\ 2d list = list(reader) : reader를 통해 CSV 형식을 list로 변환, quoting
	* writer = csv. writer(f)\ writer.writerows() : 문자열 데이터를 저장하는 객체를 반환, newline
	* dic_reader = csv.DictReader(f)\ dict_list = list(dic_reader) : 딕셔너리(dict) 형태로 한 줄씩 반환, restval, fieldnames
	* dic_writer = csv.DictWriter(f)\ dic_writer.writerows() : 딕셔너리(dict) 데이터를 CSV 파일로 저장, writeheader()

4. json 파일 입출력
- 작성 문법
	with open('파일 경로','r/w/a','encoding) as f:
	* 딕셔너리 리스트 - dict_list = json.dump(dict_list, f, ensure_ascii=False, indent=)
	* 딕셔너리 리스트 - dict_list = json.load(f)
	* 객체 리스트 - ins_list = json.dump(ins_list, f, default=Class_name, func_name)
	* 객체 리스트 - json.load(f, object_hook=func_name)

5. hdf5/h5 파일 입출력
- 딥러닝 모델 저장, 데이터 프레임 입출력 용도(확장자 : h5, keras)
- 작성 문법
	import pandas as pd
	* data_frm = pd.read_hdf('파일 경로', key) : h5(hdf/hdf5)파일을 데이터 프레임로 불러오기
	* data_frm.to_hdf('파일 경로', key) : 데이터 프레임을 h5(hdf/hdf5)파일로 저장하기
