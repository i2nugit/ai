5. 연산자
	* 산술연산자 : +, -, *, /, //(몫), %(나머지), **(제곱)
	* 할당연산자 : =, +=, -=, *=, /=, //=, %=
	* 논리연산자 : and(그리고), or(또는), not(부정)
	* 비교연산자 : >, >=, <, <=, ==(같다), !=(다르다)
		- 연산자 우선순위 규칙 : 비트 연산자(&) >> 비교 연산자 >> 논리 연산자(and/or)
	* 비트연산자 : &, |, ^(배타적 논리합/XOR), ~(비트 반전/NOT)

6. 문자열
	* 기본 제어 : 인덱싱, 슬라이싱, len(문자변수) (문자열 길이 조회)
	* dir(str) : 문자열에서 사용 가능한 모든 내장 함수 목록 확인
	* 주요 문자열 함수
		- replace(old, new) : 기존 문자(old)를 새 문자(new)로 전체 치환
		- find() : 특정 문자가 시작하는 첫 번째 인덱스 반환 (없으면 -1 반환, 에러 없음)
		- split() : 공백 또는 지정한 구분자를 기준으로 문자열을 쪼개어 리스트로 반환
		- strip() : 문자열 좌우의 공백이나 특정 지정 문자를 제거
		- join() : 리스트 등의 문자열 요소들을 특정 문자로 연결하여 하나의 문자열로 결합
		- upper() / lower() : 대문자 / 소문자 변환
		- startswith() / endswith() : 특정 문자로 시작하는지 / 끝나는지 여부 판별 (True/False)

7. 시간과 날짜 다루기 (datetime 모듈)
	* datetime : 년, 월, 일, 시, 분, 초 정보를 모두 포함하는 구조
		- datetime.now() : 함수 실행 현재 시점의 날짜와 정확한 시각 반환
	* date : 년, 월, 일 정보만 다루는 구조
	* time : 시, 분, 초 정보만 다루는 구조 (※ 날짜 정보가 없으므로 변수 간 빼기(-) 연산 불가)
	* 날짜 연산 규칙 : datetime 객체끼리 빼기(-) 연산을 수행하면 두 날짜의 시간 간격을 나타내는 timedelta 객체가 반환됨

※ 정규 표현식 간략 문법
	\d : 숫자와 매치 ([0-9]와 동일)
	\D : 숫자가 아닌 모든 문자와 매치
	\w : 문자 및 숫자와 매치 ([a-zA-Z0-9_] 및 한국어/유니코드 문자 포함)
	\W : 문자 및 숫자가 아닌 특수문자나 공백 등과 매치
	.  : 줄바꿈 문자(\n)를 제외한 임의의 문자 딱 1개와 매치
	\. : 순수한 문자로서의 마침표(.) 기호 자체와 매치
	\s : 공백 문자(스페이스, 탭, 줄바꿈 등)와 매치
	?  : 앞의 패턴이 0회 또는 1회 반복 (존재하거나 존재하지 않음)
	+  : 앞의 패턴이 최소 1회 이상 연속으로 반복
	* : 앞의 패턴이 0회 이상 반복 (없어도 되고 여러 개 있어도 됨)
	{n}  : 앞의 패턴이 정확히 n회 연속 반복
	{n,} : 앞의 패턴이 최소 n회 이상 연속 반복 (예: 특정 자리 이상 연속된 데이터 분류)
	{n,m}: 앞의 패턴이 최소 n회에서 최대 m회까지 연속 반복

※ 정규 표현식 온라인 연습장 : https://regexr.com/
	- ex1. 일반적인 전화번호 패턴 추출 예시 : [0]\d{1,2}[\-)\s\.]\d{3,4}[\-)\s\.]\d{4}
	- ex2. 계정명@도메인 형태의 이메일 패턴 추출 표준 답안 : \w+@\w+(\.\w+){1,2}
