5장 함수

1. 함수의 정의 및 사용
	- def 함수명(매개변수..):
	- 오버로딩 불가
		오버로딩 : 매개변수만 다른 같은 이름의 함수
	- docstring : 함수 첫 줄의 설명서 역할
	- 지역변수와 전역변수
		변수의 값을 조회할 때, 로컬 심볼 테이블 - 전역 심볼 테이블 - 내장 이름 테이블 순서로 조회
		global을 활용하여 함수 내에서 전역변수 사용 가능
	- 값에 의한 호출 : 스칼라 변수
	- 참조에 의한 호출 : 리스트, 튜플, 셋, 딕셔너리, 객체
	- 함수의 이름 변경
		함수형 언어 파이썬 - 변수와 함수는 동일하게 취급

2. return
	- 함수의 실행 결과를 반환함
	- 반환 값을 함수의 결과를 호출한 위치에 전달하기 위해 사용
	- 여러 값을 반환 시 : 튜플

3. 함수의 매개변수
	- 매개변수에 기본값을 가질 때 함수 호출 시 선택적으로 명시
		-  기본변수가 스칼라 변수(문자, 숫자, 논리형)일 때
			기본값은 함수가 정의되는 시점 기준
		- 기본변수가 리스트, 튜플, 셋, 딕셔너리 또는 객체일 때
			함수가 호출되는 시점의 item이 기본값이 된다
	- 순서인수 : 함수에 정의(입력)된 순서대로 전달하는 방식
	- 키워드인수 : 인수의 이름을 명시하는 방식
	- 튜플 매개변수 : *args ; 가변 인수를 구현할 때 사용
	- 딕셔너리 매개변수 : **args ; 여러 개의 키워드 인수를 딕셔너리 형태로 사용
	- 매개변수의 순서 : 순서 인수 >> 튜플 인수(*) >> 키워드 인수(기본값) >> 딕셔너리 인수(**)
	- 인수 언패킹 : 함수 호출 시 매개변수 앞 *(튜플 인수 언패킹)/ ** (딕셔너리 인수 언패킹)

4. 람다식
	- lambda [매개변수] : [리턴값]
	- 함수의 매개변수로 / 함수를 반환할 때 사용
		- map(func, vector data) # vector data : list, tuple, 배열, 시리즈 - 일차형(선형) 데이터
		- filter(func, voctor data)
		- sort(key=func_sort, reverse=T/F) # func_sort : 정렬 기준이 될 데이터를 반환하는 함수
