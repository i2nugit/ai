1. 변수
	* 변수명 명명 규칙 :
		- 반드시 문자로 시작해야 함 (숫자로 시작 불가)
		- 문자와 숫자, 언더바(_) 기호만 변수명으로 사용 가능 (특수문자 및 공백 불가)
		- 파이썬 내장 예약어(for, if, while, def 등)는 변수명으로 절대 사용 불가
	* 주의사항 : 
		- 내장 함수명(print, str, int 등)은 변수명으로 덮어써서 사용할 수 있으나 시스템 에러를 유발하므로 절대 지양함 (사용이 꼭 필요할 경우 뒤에 언더바를 붙여 str_ 형태로 사용)
	* 다중 변수 할당 : 여러 변수에 동시에 값을 일괄 할당 가능 (예: a = b = c = 0 또는 a, b = 1, 2)

2. 화면 입출력 (I/O)
	* input('[안내문구]') : 사용자로부터 키보드 입력을 받는 함수 (입력칸 앞에 안내문구 라벨 추가 가능, 입력받은 모든 데이터는 기본적으로 '문자열' 타입으로 저장됨)
	* print(내용, sep='구분자', end='종결문자') : 화면에 데이터를 출력하는 함수
		- sep : 여러 내용을 쉼표(,)로 동시에 출력할 때, 내용과 내용 사이에 들어갈 구분 기호 지정 (기본값: 공백 한 칸)
		- end : 내용 출력을 마친 후 가장 끝에 붙을 종결 문자 지정 (기본값: 줄바꿈 \n)

3. 기본 자료형 및 형변환
	* 숫자형 : 
		- int(정수), float(실수)
		- complex(복소수/허수) : 수학의 i 대신 j 기호를 사용하여 표현 (예: 3 + 4j)
		- 특징 : 파이썬이 자체적으로 처리할 수 있는 정수 데이터 크기의 범위는 컴퓨터 메모리가 허용하는 한 무한대에 가까움
	* 문자열형 (str) :
		- 기본 제어 : 인덱싱(str_[idx]), 슬라이싱(str_[from:stop:step]) 지원 및 문자열 간 더하기/곱하기 연산 가능
		- 정규 표현식 연계 핵심 함수 (re 모듈) :
			- re.findall(패키지/패턴, 문자열) : 패턴과 일치하는 모든 부분을 찾아 리스트로 반환
			- re.search(패턴, 문자열) : 패턴과 일치하는 첫 번째 매칭 정보를 객체로 반환
			- re.sub(패턴, 바꿀문자, 문자열) : 패턴과 일치하는 부분을 다른 문자로 치환/삭제
	* 논리형 (bool) : True(참) 또는 False(거짓) 값만 가짐
		- 파이썬에서 거짓(False)으로 취급하는 데이터 종류 : 숫자 0, 빈 문자열 '', 빈 리스트 [], 빈 튜플 (), 빈 딕셔너리/세트 {}, 데이터 없음 None
	* 자료형 확인 및 형변환 함수 :
		- type(변수) : 현재 변수의 자료형을 확인
		- int() : 데이터 타입을 정수형으로 강제 변환
		- float() : 데이터 타입을 실수형으로 강제 변환
		- str() : 데이터 타입을 문자열형으로 강제 변환

4. 포맷팅 (출력 형식 지정)
	* 정의 : 출력할 문자, 숫자, 날짜/시간 데이터에 서식과 자릿수를 지정하여 가독성을 높이는 방법
	* 구식 포맷팅 (% 연산자) : 코드가 길어지면 작성하기 불편하고 가독성이 떨어져 현재는 잘 쓰이지 않음
		- %s : 문자열 대입, %d : 10진수 정수 대입, %f : 실수 대입
	* format() 함수 포맷팅 : 문자열 내에 {} 중괄호를 두고 뒤에서 변수를 매핑하는 방식
		- 인덱스 번호 지정 가능 : {0}, {1} 등 중괄호 안에 번호를 적어 format() 함수 내부 매개변수의 순서를 직접 지정하여 다중 출력 가능
	* 날짜 및 시간 서식 지정 지정 코드 (strftime 활용 시 사용) :
		- %Y : 연도 4자리 (Year)
		- %m : 월 2자리 (Month)
		- %d : 일 2자리 (Day)
		- %H : 시 2자리 (Hour)
		- %M : 분 2자리 (Minute)
		- %S : 초 2자리 (Second)
