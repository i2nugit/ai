1. matplotlib

1.1 데이터 준비 및 기본설정 
1.2 그래프 그리기 기본 단계
	- 시각화 단계 : 데이터 준비 -> 그래프 객체 생성 -> 그래프 함수 실행 -> 커스터마이징 -> 저장 및 출력
1.3. 그래프 영역 나누기 : subplot, subplots
1.4 다양한 그래프 그리기 : df.plot(kind='line', x=None, y=None, figsize=None, title=None, grid=None, xlim=None, ylim=None)
	- kind 옵션 : line(기본), scatter, bar, barh, box, kde(밀도추정), hist(빈도) 등
1.5 그 외 그래프 그리기 : boxplot, violinplot, bar, barh, hist, fill_between, axvline, axhline, pie
1.6 그래프 커스터마이징 : text(),set_title(), annotate(xytext, xy,arrowprops=dict(arrowstyle, connoctionstyle,color)), subplots_adjust(hspace, wspace)
