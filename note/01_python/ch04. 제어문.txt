4장. 제어문

- 1. 조건문 (if / elif / else)
	* 특징 : 코드 블록을 구분하기 위해 '들여쓰기'와 블록의 시작을 알리는 '콜론(:)'이 필수임
	* 조건의 조건 : 조건식의 결과는 반드시 논리형(True/False)으로 판별될 수 있어야 함
	* 구조 :
		- if 조건1:
			조건1이 True일 경우 수행할 명령어들 (if문은 단독 사용 가능)
		- elif 조건2:
			조건1이 거짓이고, 조건2가 True일 경우 수행할 명령어들 (개수 제한 없이 추가 가능)
		- else:
			위의 모든 조건이 False일 경우 수행할 명령어들 (단독 사용 불가, 반드시 if문 뒤에 위치)
	* 조건부 표현식 : [참일 때 값] if [조건식] else [거짓일 때 값] (예: score = 'A' if total >= 90 else 'B')

- 2. 반복문 (for / while)
	* for-in문 : for 임시변수 in 나열_가능한_자료 :
		- 시퀀스(리스트, 튜플, 문자열 등)나 반복 가능한 객체의 요소를 처음부터 끝까지 순회한 후 종료
	* while문 : while 조건식 :
		- 조건식이 True인 동안 블록을 무한 반복하며, 조건식이 False가 되는 순간 종료
	* 반복문과 else 구조 (for-else / while-else) :
		- 루프 정상 종료 시 실행 : 반복문이 break를 만나지 않고 끝까지 완벽하게 정상 순회/종료되었을 때만 else 블록이 실행됨
	* 흐름 제어 키워드 :
		- break : 현재 break를 포함하고 있는 가장 가까운 반복문 하나를 즉시 강제 종료하고 탈출
		- continue : 반복문 내에서 continue 이후의 남은 문장들을 건너뛰고 다음 루프(블록)로 이동

- 3. 난수 생성 random
	* 사용법 : import random 선언 후 호출
	* 주요 함수 :
		- random.randint(low, high) : low 이상 high '이하'의 임의의 정수 하나를 반환
		- random.choice(sequence)   : 리스트, 튜플 등 시퀀스 자료형에서 임의의 요소 하나를 무작위 추출
		- random.sample(population, k) : 모집단(시퀀스/셋)에서 중복 없이 k개의 요소를 리스트 형태로 무작위 샘플링

- 4. 중첩 루프 및 다중 루프 탈출
	* 중첩 루프 : 반복문 내부에 또 다른 반복문이 포함된 구조 (예: 구구단, 2차원 공간 순회)
	* 중첩 루프 탈출 방법 1 (Flag 변수 활용) :
		- 외부 루프와 내부 루프가 공유하는 논리형 변수(예: is_break = False)를 선언하여, 
		  내부에서 조건 만족 시 Flag를 True로 바꾸고 break -> 외부 루프에서 Flag를 검사하여 다시 break하는 방식
	* 중첩 루프 탈출 방법 2 (Exception 예외처리 활용) :
		- 전체 중첩 루프를 try 블록으로 감싸고, 가장 안쪽 루프에서 조건 만족 시 
		  raise 사용자_정의_예외를 강제 발생시켜 바깥쪽 except 블록으로 단번에 건너뛰어 탈출하는 방식
