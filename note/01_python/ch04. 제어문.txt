4장. 제어문

1. 조건문(if/elif/else) : 들여쓰기(블록), 콜론(블록의 시작)
- 조건은 반드시 True/False로 판별될 수 있어야함

- if 조건 1 :
    조건 1이 참일 경우 수행할 명령어들(if문 구절 단독 사용 가능, 들여쓰기 필수)
- elif 조건 2 :
    조건 1이 거짓이고, 조건 2가 참일 경우 수행할 명령어들(필요한 만큼 추가 사용 가능)
- else :
    모든 조건이 거짓일 경우 수행할 명령어들(단독으로 사용 불가; if 구절 필수)

- 조건식 : 함수('A' if 조건 else 'B') ; 조건이 True일 경우 A, False일 경우 B

2. 반복문
- for-in문 : for 임시 변수 in 나열 가능한 자료/else(생략가능)
			나열 가능한 자료가 없을 경우 for문 종료
- while문 : while 조건/else(생략가능)
			조건이 False가 될 경우 while문 종료
- 난수 : import random, randint(),choice(),sample()
- break : 반복문 진행 중, break 시 break를 포함하는 반복문을 종료
- continue : 반복문 내에서 continue 시 이후의 문장을 건너뜀
